# مشروع تحليل المشاعر للتعليقات العربية (Sentiment Analysis)

هذا المشروع يهدف إلى بناء نموذج ذكاء اصطناعي لتحليل التعليقات العربية وتصنيفها إلى:

* **إيجابي (positive)**
* **سلبي (negative)**
* **محايد (neutral)**

يتم ذلك باستخدام بايثون وتقنيات **معالجة اللغة الطبيعية (NLP)**.

---

# المكتبات المطلوبة

تثبيت المكتبات عبر:

```
pip install -r requirements.txt
```

---

# تنظيف النصوص العربية

يتم تنظيف النصوص عبر ملف:

```
src/clean_text.py
```

ويتضمن:

* إزالة الرموز غير العربية
* إزالة الأرقام والعلامات
* حذف الكلمات التوقفية (Stopwords)
* تنسيق النص

---

# تدريب نموذج المشاعر

لتدريب النموذج لأول مرة:

```
cd src
python train_model.py
```

عند الانتهاء سيتم إنشاء ملفات:

```
models/sentiment_model.pkl
models/tfidf_vectorizer.pkl
```

---

# استخدام النموذج للتوقع

## من خلال ملف predict.py

```
python predict.py
```

يقوم بعرض أمثلة جاهزة للتصنيف.

---

## من خلال CMD (الأفضل)

شغّل:

```
python cli_predict.py
```

ثم اكتب تعليقك في الطرفية مثل:

```
المنتج ممتاز جداً وخدمة فنتاستكيه
➡ التصنيف: positive
```

للخروج:

```
exit
```

---


# وصف المشروع

يعتمد المشروع على:

* تنظيف النصوص العربية
* تحويل النص إلى أرقام باستخدام TF-IDF
* تدريب نموذج Logistic Regression
* تقييم النموذج عبر الدقة و F1-score
* استخدام النموذج لتوقع مشاعر أي تعليق عربي
